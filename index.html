Frequently Asked Questions

What is this?

ZeroLogDNS is a hobby non-profit project which is not logging any DNS query information.

What does the ZeroLogDNS servers logging?

There is absolutely no logging! I keep logs only of the traffic bandwidth. Your IP and Query log is very simply directed in to /dev/null (/dev/null is a black hole in Linux systems)

How can i use this service?

This service does not offer plain-text-dns. If you want to use this service you need a DoT/DoH client. You can use DoH simply from your browser like Firefox or Chrome.
